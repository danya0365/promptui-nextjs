# promptui — Master Build Prompt

สร้างโปรเจค **promptui** ด้วย Next.js (App Router) + Tailwind CSS + TypeScript

---

## ภาพรวม

promptui คือ UI Showcase Gallery แบบ full-screen สำหรับแสดง UI component ตัวอย่าง
พร้อม prompt ที่ใช้สร้าง UI นั้น ๆ เพื่อให้ผู้ใช้ดูตัวอย่าง แล้ว copy prompt ไปใช้กับ AI ได้ทันที
